# 뛰장 사용자 앱 기능 가이드

> 대분류 5가지 — 리그 / 매치 / 게스트 / 마이페이지 / 팀페이지

---

## 1. 리그

풋살 리그 정보를 탐색하고, 팀 운영자는 리그에 참여 신청할 수 있는 섹션입니다.

### 1-1. 리그 메인
- **배너 슬라이드** — 운영자가 등록한 이벤트/리그 배너를 슬라이드로 표시. 배너 클릭 시 연결된 리그 상세 또는 외부 링크로 이동
- **다가오는 리그 경기** — 배너가 없을 때 표시. 임박한 리그 경기(홈팀 vs 원정팀, 날짜·시간·구장)를 슬라이드로 확인. 카드 클릭 시 매치 상세 이동
- **리그 참가 팀** — 현재 진행 중인 리그에 소속된 팀 목록(팀 로고, 리그 배지). 클릭 시 해당 팀페이지 이동
- **팀 회원 모집** — 현재 회원을 모집 중인 팀 목록. 클릭 시 모집 상세 이동
- **리그 일정 / 리그 현황** 버튼 — 하단 패널에서 서브 뷰로 전환

### 1-2. 리그 일정
- **리그 선택** — 진행 중인 리그를 드롭다운으로 선택
- **월 이동** — 이전/다음 달 탐색
- **예정 경기 목록** — 선택한 리그의 해당 월 예정(SCHEDULED) 경기 카드 목록 (날짜·시간·구장·홈/원정팀)

### 1-3. 리그 현황
- **리그 선택** — 내 팀이 참가 중인 리그만 표시
- **순위표** — 팀별 경기·승점·승무패·득실차 표시. 내 팀 행은 파란색으로 강조
- **완료된 경기 결과** — 해당 월에 종료된 경기의 점수 결과 목록

### 1-4. 리그 상세 (`/league-detail/:uid`)
- 리그 이름, 상태(모집중/진행중/완료), 시즌, 지역, 팀 수, 기간
- 운영자가 작성한 리그 소개글(HTML 렌더링)
- 참여비 안내 (캐시 차감)
- **리그 참여하기** 버튼 — 팀 운영자만 신청 가능. 참여비가 있을 경우 캐시 차감 확인 후 신청

## 2. 매치

내 팀의 경기 일정을 관리하고, 매치 상세 정보를 확인하는 섹션입니다.

### 2-1. 매치 메인
- **팀 없는 사용자**
  - 팀 코드 입력 후 **팀 가입하기** — 팀 코드로 팀을 검색해 모집 상세 페이지로 이동
  - **팀 만들기** — 팀 생성 플로우(이름→마크 업로드→지역 설정)로 이동
  - 가입 대기 중일 경우 안내 메시지 표시 및 신청 현황 페이지 이동 링크
- **팀 있는 사용자**
  - 본인 팀 코드 표시 및 **팀 입장하기** 버튼
  - **월별 달력** — 날짜 선택으로 해당 날 매치만 필터링. 경기가 있는 날짜에 점 표시
  - **매치 일정 목록** — 친선경기/자체경기, 경기 포맷(4v4~7v7), 상태(모집중·매칭완료·경기종료 등), 날짜·구장·참여 인원

### 2-2. 매치 상세 (`/match-detail/:uid`)

| 매치 유형 | 주요 표시 정보 |
|-----------|---------------|
| 리그 매치 | 홈/원정팀 로고·점수, 팀 회원수·평균나이·전체전적·리그전적 |
| 친선 경기 | 주최팀, 성별·연령대·경기시간·경기수·매너점수·리그등급 |
| 자체(자유) 경기 | 주최자, 성별·연령대·경기시간·포지션(필드/GK) |
| 게스트 모집 | 모집팀·매너점수, 동일 친선 항목 + 모집 포지션·참가비·보장시간·모집인원 |

**공통 기능:**
- 참여자 명단 (프로필 사진/이름)
- 아코디언 섹션: 구장 정보 / 매치 규칙 / 추가 사항 / 모집 정보(게스트만) / 리그 규칙(리그만)
- **신청하기** — 무료 경기: 확인 후 신청. 유료 경기: 캐시 잔액 확인 → 차감 동의 후 신청
- **경기종료** — 경기 후 팀 운영자가 결과 입력(홈·원정 점수). 입력 후 매너 점수 모달로 연결
- **매너점수 입력** — 경기 완료 후 참여자가 상대팀에게 별점 1~5점으로 평가

### 2-3. 매치 일정 만들기 (`/match-create`)
- 친선경기 또는 자체경기 유형 선택
- 경기 포맷, 날짜, 시간, 구장, 참가비, 경기 규칙 등 설정

## 3. 게스트

팀 내 빈자리에 외부 게스트를 모집하거나, 개인이 게스트로 경기에 참여할 수 있는 섹션입니다.

### 3-1. 게스트 메인
- **추천 매치 슬라이드** — 현재 모집 중인 게스트 매치를 카드로 표시 (포지션·매너점수·참가비·일시·위치·현재인원/최대인원)
- **날짜 달력** — 월 이동 및 날짜 선택으로 해당 날 모집 목록 필터링. 게스트 있는 날짜에 점 표시
- **게스트 목록** — 모집완료된 항목은 흐릿하게 처리 + "모집완료" 뱃지 표시
- 카드 클릭 시 게스트 매치 상세 페이지 이동

### 3-2. 게스트 신청 (매치 상세)
- 게스트 상세에서 **게스트 신청하기** 버튼
- 참가비가 있는 경우 보유 캐시 확인 후 캐시 차감 동의
- 신청 완료 후 참여자 명단에 표시

### 3-3. 게스트 모집 등록 (`/guest-recruit`) — 팀 운영자 전용
3단계 플로우로 진행:

**Step 1. 매치 선택**
- 7일 이내 팀 매치 일정 목록 표시
- 이미 게스트 모집 중인 매치는 선택 불가(뱃지 표시)
- 매치 일정이 없으면 "매치 일정 먼저 만들기" 링크 제공

**Step 2. 모집 조건 설정**
- 모집 인원 (경기 포맷에 따라 최대 인원 자동 계산)
- 성별 (남자 / 여자 / 남녀무관)
- 나이 (10대~50대 이상 / 나이무관)
- 포지션 (필드 / 골키퍼)
- 참가비 (무료 / 5,000 / 8,000 / 10,000원)
- 보장시간 (30 / 60 / 80 / 100분)

**Step 3. 추가 안내사항 입력 및 등록**
- 게스트에게 전달할 주차·유니폼·집합장소 등 자유 입력
- 안내사항: 매치 시작 1시간 전까지 취소 가능, 무단 불참 패널티 안내

## 4. 마이페이지

개인 프로필, 캐시, 활동 내역을 관리하는 섹션입니다.

### 4-1. 프로필 영역
- 이름 표시 및 **프로필 수정** 버튼 → 이름·연락처·생년월일 등 수정 가능

### 4-2. 통계 카드
| 항목 | 내용 |
|------|------|
| 매너점수 | 상대팀에게 받은 별점 평균 |
| 소속 팀 / 나의 팀 | 가입된 팀 이름. 클릭 시 팀페이지 이동 |
| 참여 경기수 | 신청한 매치 중 완료된 경기 수 |

### 4-3. 나의 캐시
- 현재 캐시 잔액 표시
- **캐시 충전** 버튼 → 충전 금액 선택 후 결제(PG 연동)

### 4-4. 메뉴 목록

| 메뉴 | 설명 |
|------|------|
| 뛰장 소개 | 서비스 소개 페이지 |
| 신청 내역 | 내가 신청한 매치·리그 목록 (상태별 확인, 취소 가능) |
| 캐시 사용내역 | 충전·사용·환불 거래 내역 조회 |
| 구단 후원 신청 | 팀 구단주 후원 신청 폼 |
| 자주 묻는 질문 | FAQ |
| 공지사항 | 운영 공지 |
| 뛰장 가이드 | 매너점수 시스템 및 서비스 이용 가이드 |
| 로그아웃 | 확인 후 로그아웃 |

### 4-5. 신청 내역 (`/my-applications`)
- **매치 탭** — 신청한 매치 목록 (주최팀·구장·날짜·유형·상태). 취소 가능한 항목에 "신청 취소" 버튼 표시
- **리그 탭** — 신청한 리그 목록

## 5. 팀페이지

팀의 커뮤니티, 경기, 리그, 회원 정보를 통합 관리하는 섹션입니다.

### 5-1. 팀 프로필 헤더
- 팀 로고, 이름, 운영자명, 내 역할(운영자/매니저/일반회원) 표시
- 운영자 전용: **대시보드** / **팀 설정** 버튼
- **구단주 정보 바** — 현재 구단주 이름 및 정보 확인

### 5-2. 커뮤니티 탭
- **공지사항** — 운영자가 등록한 공지(상단 고정, 클릭 시 내용 펼침)
- **일반 게시글** — 팀원 작성 글 피드 (제목·내용·이미지)
  - 좋아요 (별 아이콘, 클릭으로 토글)
  - 댓글 달기 / 댓글 삭제(내 댓글)
  - 작성자 본인 / 운영자: 수정·삭제 가능
- **글쓰기 버튼** — 팀원 이상이면 새 게시글 작성 가능

### 5-3. 매치경기 탭
- **필터** — 전체 / 친선경기 / 자체경기
- **매치 카드** — 경기 포맷·유형 뱃지, 날짜·구장, 참여인원/최대인원
- 운영자·매니저: 모집중인 매치에 수정·삭제 메뉴 표시
- 카드 클릭 시 매치 상세 이동

### 5-4. 리그경기 탭
- **리그 선택** — 참가 중인 리그가 여러 개면 드롭다운 선택
- **순위표** — 전체 팀 순위(경기·승점·승무패·득실차). 내 팀 행 강조
- **액션 버튼** — 전체 일정 / 리그 현황 / 팀 리그 현황 페이지 이동
- **팀 리그 일정** — 내 팀이 참가하는 경기 목록 (날짜·구장·홈vs원정·점수)

### 5-5. 회원정보 탭
- 전체 회원수, 성별(남/여) 인원, 평균 연령대
- 회원 목록 (이름·역할·포지션 등)

### 5-6. 팀 대시보드 (`/team-dashboard`) — 운영자 전용
| 영역 | 항목 |
|------|------|
| 리그 대시보드 | 리그 등급(순위), 전적(승무패), 승률, 승점·득실차, 매너점수 |
| 매치 대시보드 | 전체 경기수, 매치 전적 |

### 5-7. 팀 설정 (`/team-settings`) — 운영자 전용
- 팀 프로필 수정 (이름·소개·지역·마크 이미지)
- 가입 신청 승인 관리 — 신청자 목록, 승인/거절
- 회원 관리 — 역할 변경, 강제 탈퇴
- 팀 삭제 요청

### 5-8. 팀 가입 플로우 (회원 모집)
- `/team-recruit` — 회원 모집 중인 팀 목록 탐색
- `/team-recruit-detail/:teamUid` — 팀 상세 및 가입 신청
- `/pending-approval` — 가입 신청 후 대기 현황 확인
- `/pending-manage` — 운영자의 가입 신청 승인/거절 관리
